# Bet — Gen-Z Accountability App

Built at the **8x hackathon** ([event](https://luma.com/t4j6h0br)) — a consumer mobile app in 4 hours.

## What it is

Stake **real money** on a personal goal:

- Hit the goal → get the money back.
- Miss it → lose it (to a pool / charity / friends).

The core hook is **skin in the game** — the loss state is what makes you actually follow through. Social layer: friends see if you fail. Framed as a **commitment device, not gambling** (you bet against yourself, not others).

## Pitch (90s)

Problem → Solution → Demo → Close. Demo-weighted: ~50s on the live demo, with the **loss state** as the emotional centerpiece.
